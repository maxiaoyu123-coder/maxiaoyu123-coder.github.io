groovy

关键字

总结内容

变量名规范：大小写/数字/下划线/驼峰

动态类型

第一次申明def

引用：变量名使用对象的内存空间

双三引号解释性字符串，单引号字面

赋值：

  • 普通赋值 def a = 10
  • 多次赋值 def a=10 b=20
  • 链式赋值 a=b=10
  • 自操作赋值 a++ b--
  • 解压赋值 (a,b,c,d,e) = [1,2,3,4,5]
  • 赋NULL值 def a 等价 def a=null

字符串数据类型

  • 索引/索引段截取：0第一、-1最后、[6..-1] 第7-最后、[6..-2] 第7-倒数第2
  • +、-、* +和-只执行一次，如：
    • 'abcad' - 'a' 得'bcad'(从左开始找并删然后停)，
    • 'abcad' + 'a' 得'bcad'(从右后加)
  • 计算长度/是否为空字符串/统计子串次数/判断包含/判断结尾/摘除缩进/定位子串位置/大小写/切割为列表(两种)
  • 正则
    • 三个操作符：=~ ==~ ~
    • 采用正则的字符串方法：
      • split/tokenize
      • replaceAll
      • eachMatch

列表

  • 有序保存有序占用
  • 截取[]: 单个/多个/范围
  • 赋值元素(2种)
  • + - *
    • - 是删除列表中全部相同的元素
    • [1,2,3]+[4,5] 列表元素并入右后
  • 判断包含/计算长度(2两种)/是否为空列表/嵌套列表并入/交际/逆序(索引)/排序(值)
  • pop/remove/add/unique
  • 支持闭包的方法：findAll/find/collect

映射

  • 赋值 info = ['name': 'John', 'age': 35]
  • 无序/索引key可为任何类型(含列表)
  • 截取, 映射['key']或映射.key
  • 计算大小/列出全部key到列表/列出全部值到列表

范围

  • 一个的整数区间
  • 赋值 range_test = (1..10)
  • 截取[]： 单个/多个/范围
  • 方法：是否包含/
  • 支持闭包的方法：each/
  • 搭配for循环：for (i in (1..10))

方法

  • 实参(外部的变量)与方法内的形参指向不同对象的内存地址，故方法内形参变化不影响外部的实参
  • 作用域

循环

  • for 可用列表/映射/范围
  • while 表达式

条件

  • switch
  • if/else if/else
  • 变量 = 条件表达式? 值1: 值2

闭包

  • 是一种简化的方法体，使用更便捷，可操作域外变量
  • 闭包主要搭配列表/字典/范围/字符串：
    • 字符串: eachMatch/replaceAll
    • 列表：each/find/findall/any/all/collect
    • 范围: each
