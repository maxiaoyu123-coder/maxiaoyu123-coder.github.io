GCP VM IP

  • Internal IP: Permanent Internal IP
  • External IP: Ephemeral External IP changes when VM is stopped
  • Static IP: Permanent External IP
  • External IP cant be set to Static IP

    IP can be Reserved as Static IP

  • Billed when Static IP not Used( The instance using Static IP is stopped )

How to Save Costs

Machine Types

  • Optimized combination of compute(CPU,GPU), memory, disk(storage) and networking for specific workloads
  • Create a Machine Type customized to your needs when predefined VM options are Not appropriate for your workload
  • Custom Machine Type: Adjust vCPUs, memory and GPUs
    • choose between E2, N2, N1

      Machine Types中的一般用途系列：

      • E2 最便宜，若设置为preempt可进一步降低费用
      • N2 2-80vCPU、≤640GB, 比N1效能更高

    • OS: CentOS, RedHat, Ubuntu, Debian, CoreOS, Windows etc
    • Billed per vCPU and Mem provisioned to each instance
      • Example: Hourly price $0.033174/vCPU + $0.004446/GB
